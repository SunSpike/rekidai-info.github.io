Anyone can edit the rekidai data.
If you want to update rekidai data, please fork https://github.com/rekidai-info/rekidai-info.github.io, edit rekidai.json, and submit a Pull Request. That's all.

To view beatmania IIDX SP ☆12(Lv12) rekidai scores, GitHub authentication is required.
Please agree to the terms of use.
Those who do not contribute to updating rekidai.json for a certain period of time will fail authentication.
GitHub authentication is required once a week.

beatmania IIDX SP ☆12(Lv12) の歴代スコアを見るためには、GitHub 認証が必要です。
一定期間 rekidai.json の更新に貢献しない方は認証が失敗します。
GitHub 認証は週に一度必要です。

If you want to see the table, please contribute to updating rekidai.json, even if only a little.
歴代表が見たいなら、少しでも rekidai.json の更新に貢献してください。

If you have contributed at least once to updating rekidai.json, please sign up with that GitHub account.
rekidai.json の更新に一度でも貢献したのなら、その GitHub アカウントでサインアップしてください。
